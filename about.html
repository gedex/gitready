Tentang

Jika Anda mencari petunjuk (bukan maksud sebenarnya) bagaimana menggunakan git, jangan lanjutkan! Tujuan situs ini adalah untuk membuat situs ini berguna sebagai cara untuk seorang pemula untuk segera memulai menggunakan git dan untuk menunjukkan kepada yang sudah biasa menggunakan Git beberapa trik baru.

Ingin mengganti layout, desain, atau lainnya? Lihat halaman proyek ini di Github. Ingin tip Anda mejeng di halaman depan? Kirim ke sini.

Ingin menghubungi penulis? Email ke nick@quaran.to, colek di Twitter, atau kirim pesan di GitHub.

Pernyataan mengenai tip: Penulis tidak mengklaim dirinya sempurna dalam git. Alasan utama blog ini dibuat adalah untuk belajar lebih dalam mengenai Git and bagaimana Git bekerja. Semuanya merupakan proses pembelajaran bagi yang terlibat, dan oleh karenanya sangat berarti bagi pembaca dan penulis. Jika Anda melihat kesalahan atau ketidak akuratan, jangan sungkan untuk menghubungi Nick atau tinggalkan komentar.

Dan akhirnya...terima kasih telah membaca!
