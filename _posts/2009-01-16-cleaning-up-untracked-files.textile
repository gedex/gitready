Tip pertama kiriman pembaca datang dari "Dav Glass":http://blog.davglass.com/, dan membahas perintah untuk menjaga repositori agar tetap bersih. Punya tip untuk dibagi? "Kirimkan!":http://gitready.com/submit.html

Berkas dan direktori dapat menumpuk di direktori kerja Anda, yang bisa berasal dari <em>merge</em>, hasil <em>auto generate</em>, atau mungkin tidak sengaja diletakkan. Apapun itu, Anda tidak perlu mengabaikannya via @.gitignore@, Anda cukup membuangnya. Menjalankan:

@git clean -n -d <path>@

akan melakukan '<em>dry run</em>' dari @git clean@ dan memberitahu Anda berkas dan direktori apa saja yang akan dibuang. Menjalankannya pada "gitready":http://github.com/qrush/gitready akan memberikan output:

<pre>
Would remove _posts/2009-01-16-cleaning-up-untracked-files.textile
</pre>

Yang tentunya tidak diinginkan. Saya belum menambahkan post ini ke <em>staging</em> area, jadi berkas ini belum terjejak (<em>untracked</em>). Jika hanya berkas (<em>files</em>) yang ingin dibersihkan dan bukan direktori kosong, hilangkan opsi @-d@. Jika sudah yakin ingin membuang semua berkas tersebut, menjalankan

@git clean -d <path>@ 

akan melakukan penghapusan. Jika Anda mengabaikan @<path>@, git akan melakukan penghapusan pada direktori dimana Anda mejalankannya. Mungkin Anda perlu menambahkan opsi @-f@ jika dirasa bakal ada masalah hak akses atau situasi lainnya.

Pembersihan (@git clean@) juga memiliki beberapa opsi berguna: jika Anda ingin melakukan pembersihan biasa dan membersihkan semua berkas yang biasa git abaikan, opsi @-x@ akan sangat membantu. Jika hanya ingin membuang berkas yang diabaikan saja, gunakan opsi @-X@.

Jadi untuk pembersihan keseluruhan:

<pre>
$ git clean -n -d -x
Would remove .DS_Store
Would remove _posts/.2009-01-16-cleaning-up-untracked-files.textile.swp
Would remove _posts/2009-01-16-cleaning-up-untracked-files.textile
Would remove _site/
Would remove images/.DS_Store
Would remove images/_site/
</pre>

Perintah di atas akan membersihkan berkas yang diabaikan via ".gitignore":http://github.com/qrush/gitready/blob/7f51407b796c10beb11fe9e67c0f563d41a92fd8/.gitignore dan juga berkas dan direktori lainnya yang tidak digunakan. Waspadalah saat menggunakan @git clean@, dan pastikan untuk mengecek apa yang mau dihapus.

Seperti biasa, kalau ada trik @git clean@ lainnya harap beritahu kami melalui komentar!
